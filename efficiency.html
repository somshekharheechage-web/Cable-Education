OEE Formulas

Availability = (Planned − Downtime) / Planned × 100%
Time the machine was actually running as a % of planned time.
Performance = Actual Output / Target Output × 100%
Speed efficiency — how close to target rate the machine ran.
OEE = Availability × Performance × Quality / 10000
Overall Equipment Effectiveness. World-class manufacturing target is ≥ 85%.
